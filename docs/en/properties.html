Properties

Basic Properties

this.$id Type: String

被渲染为设计器的DOM对象的id；该属性是不能为空的，需要被渲染的设计器上具有id属性。

this.$bgDiv Type: Jquery Object

设计器最外层框架的Jquery对象。

this.$tool Type: Jquery Object

设计器左侧工具栏的Jquery对象，仅初始化传参options.haveTool=true时有效；否则该属性为null。

this.$head Type: Jquery Object

设计器上方顶部栏的Jquery对象，包括标题框和顶部按钮组；仅初始化传参options.haveHead=true时有效；否则该属性为null。

this.$workArea Type: Jquery Object

装载结点/线条/分组泳道的绘图工作区，Jquery对象。

this.$draw Type: Dom

画矢量线条的Dom容器，处于工作区中。

this.$group Type: Jquery Object

仅用来装载分组泳道DOM元素的Jquery对象容器，处于工作区中。仅初始化传参options.haveGroup=true时下有效；否则该属性为null。

this.$workExtendStep Type: Number

在手动扩展绘图工作区的可用区域时，一次扩展宽/高后增加多少像素，默认200。

this.$ghost Type: Jquery Object

专门用在移动、调整大小等元素操作时，给用户操作的半透明浮动区。仅当this.$editable=true时该属性才存在。

this.$mpFrom Type: Jquery Object

编辑模式下选中线以后，出现在起点的小框，用来移动以变更要连接的起始结点。仅当this.$editable=true时该属性才存在。

this.$mpTo Type: Jquery Object

编辑模式下选中线以后，出现在终点箭头处的小框，用来移动以变更要连接的到达结点。仅当this.$editable=true时该属性才存在。

this.$textArea Type: Jquery Object

编辑模式下双击元素后出现的浮动文本域，用来写重命名方法setName所需的新名称传参。仅当this.$editable=true时该属性才存在。

this.$lineMove Type: Jquery Object

操作移动折线的中段时用到的浮动Div。仅当this.$editable=true时该属性才存在。

this.$lineOper Type: Jquery Object

编辑模式下选定一条连线后出现的浮动操作栏，有改变线的类型和删除线等按钮。仅当this.$editable=true时该属性才存在。

Business Properties

this.$nowType Type: String

当前要绘制的对象类型，开始时为"cursor"：即不绘制任何元素，只是作为鼠标指针进行元素选定。
其它的取值还有："direct"实线连线、"dashed"虚线连线、"group"分组泳道、……各种类型的节点。

this.$focus Type: String

当前被选定的结点/转换线ID，如果没选中或者工作区被清空则为空字串。

this.$editable Type: Boolean

当前工作区是否可编辑，即是编辑模式还是仅浏览模式。

this.$nodeDom Type: JsonMap

流程图中所有展示节点的Jquery对象Map集；以节点id为key，工作区中节点元素的Jquery对象为value。

this.$lineDom Type: JsonMap

流程图中所有展示连线的Jquery对象Map集；以连线id为key，工作区中连线元素的Dom对象为value。

this.$areaDom Type: JsonMap

流程图中所有展示分组泳道的Jquery对象Map集；以分组泳道id为key，工作区中分组泳道元素的Jquery对象为value。

this.$scale Type: Number

工作区内流程图当前相对于原始大小的缩放比例，默认为1（100%），范围是0.5~4。

this.$headBtnEvents Type: JsonMap

用户对头部栏另行自定义类型按钮的事件绑定map集合，以按钮类型名为key，对应的function类型变量为value。

this.$undoStack Type: Array[Json Object]

“撤销操作”事务栈；仅当初始化传参options.haveTool=true且options.useOperStack=true时该属性才存在。

this.$redoStack Type: Array[Json Object]

“重做操作”事务栈；仅当初始化传参options.haveTool=true且options.useOperStack=true时该属性才存在。

this.$isUndo Type: Number

事务操作标志位，仅供设计器运作时内部调用。

Data-store Properties

this.$title Type: String

被载入的流程图名称；如果初始化时设定有有顶部栏和标题框，则显示于标题框中。

this.$nodeData Type: JsonMap

载入的流程图中所有节点数据Map集；以节点id为key，NodeItem类型结构的数据为value。
详见“数据结构定义”章节中的“NodeItem”介绍。

this.$nodeCount Type: Number

载入的流程图中节点的数量，初始为0。

this.$lineData Type: JsonMap

载入的流程图中所有连线数据Map集；以连线id为key，LineItem类型结构的数据为value。
详见“数据结构定义”章节中的“LineItem”介绍。

this.$lineCount Type: Number

载入的流程图中连线的数量，初始为0。

this.$areaData Type: JsonMap

载入的流程图中所有分组泳道数据Map集；以分组泳道id为key，AreaItem类型结构的数据为value。
详见“数据结构定义”章节中的“AreaItem”介绍。

this.$areaCount Type: Number

载入的流程图中分组泳道的数量，初始为0。

this.$max Type: Number

计算默认元素名称中，后缀序列号的SEQUENCE，默认值为1；
初始值来自构造方法中options传参的initNum属性，或者载入流程图JSON数据中的initNum属性；
当有流程图中有元素被手动新增时，该值会自加1。

this.$deletedItem Type: JsonMap( {id: String )

在流程图的编辑操作中被删除掉的元素id-map集合；
以元素id为Key，元素类别名为value(共有node'、'line'、'area'三种取值)。

this.$extra Type: Extra

载入的流程图中的附加数据Map集，不直接参与流程图的构建，输出流程图数据时将原样返回。 详见“数据结构定义”章节中的“Extra”介绍。
