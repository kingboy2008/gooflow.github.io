Properties

Basic Properties

this.$id Type: String

The id of the DOM object rendered as the designer; this property cannot be empty, and the designer that needs to be rendered has the id attribute on it.

this.$bgDiv Type: Jquery Object

The Jquery object of the designer's outermost frame.

this.$tool Type: Jquery Object

The Jquery object on the left side of the designer's toolbar, which is valid only when the parameter is initialized with options.haveTool=true; otherwise the property is null.

this.$head Type: Jquery Object

The top-column Jquery object above the designer, including the title box and the top button group; it is valid only when the argument is initialized with options.haveHead=true; otherwise the property is null.

this.$workArea Type: Jquery Object

Load the node/line/group lane drawing workspace, Jquery object.

this.$draw Type: Dom

Paint a vector line of Dom containers in the workspace.

this.$group Type: Jquery Object

A Jquery object container that is only used to load grouped swimlane DOM elements in the work area; it is valid only when the argument is initialized with options.haveGroup=true; otherwise the property is null.

this.$workExtendStep Type: Number

When expanding the available area of​the drawing workspace manually, how many pixels are added after expanding/width at a time, the default is 200px.

this.$ghost Type: Jquery Object

A translucent floating area that is used by the user to operate on elements such as moving and resizing. Only if this.$editable=true the property is only present when.

this.$mpFrom Type: Jquery Object

After the line is selected in edit mode, the small box at the end point arrow is used to move to change the arrival node to be connected. Only if this.$editable=true the property is only present when.

this.$mpTo Type: Jquery Object

After the line is selected in edit mode, the small box at the end point arrow is used to move to change the arrival node to be connected. Only if this.$editable=true the property is only present when.

this.$textArea Type: Jquery Object

The floating text field that appears after double-clicking an element in edit mode is used to write setName the new name pass arguments required by the rename method . Only if this.$editable=true the property is only present when.

this.$lineMove Type: Jquery Object

The floating Div used when manipulating the midline of a moving polyline. Only if this.$editable=true the property is only present when.

this.$lineOper Type: Jquery Object

The floating Div used when manipulating the midline of a moving polyline. Only if this.$editable=true the property is only present when.

Business Properties

this.$nowType Type: String

The current type of object to be drawn starts with "cursor": no element is drawn, but is selected as a mouse pointer.
Other values include: "direct" solid lines, "dashed" dotted lines, "group" grouping lanes, …… various types of nodes.

this.$focus Type: String

The currently selected node/conversion line ID, if not selected or the workspace is empty, it will be an empty string.

this.$editable Type: Boolean

Whether the current workspace is editable, that is, edit mode or browse mode only.

this.$nodeDom Type: JsonMap

Map Jquery objects set flowchart showing all nodes; takes Node's id as key, the node's Jquery Object element as value.

this.$lineDom Type: JsonMap

Map Jquery objects set flowchart showing all lines; takes Line's id as key, the node's Dom Object element as value.

this.$areaDom Type: JsonMap

Map Jquery objects set flowchart showing all lanes; takes Lane's id as key, the lane's Jquery Object element as value.

this.$scale Type: Number

The scale of the flowchart in the workspace is currently relative to the original size. The default is 1 (100%) and the range is 0.5~4.

this.$headBtnEvents Type: JsonMap

Users of the head of field additionally customize the type of event button bindings map collections to button type name as a key, corresponding function type variableto value.

this.$undoStack Type: Array[Json Object]

The transaction stack of "undo"; only when initialization parameter passing options.haveTool=true and options.useOperStack=true when the property is only present.

this.$redoStack Type: Array[Json Object]

The transaction stack of "redo"; only when initialization parameter passing options.haveTool=true and options.useOperStack=true when the property is only present.

this.$isUndo Type: Number

The transaction operation flag, which is only called internally by the designer when it operates.

Data-store Properties

this.$title Type: String

The name of the flowchart to be loaded; if there is a top bar and a title bar when it is initialized, it is displayed in the title box.

this.$nodeData Type: JsonMap

Loading of all the nodes in the flowchart Map data set; to node's idas a key, NodeItem data type structure to value.
For details, see "NodeItem" in the "Data Definition" chapter.

this.$nodeCount Type: Number

The number of nodes in the loaded flowchart, initially 0.

this.$lineData Type: JsonMap

Loading of all the lines in the flowchart Map data set; to line's idas a key, LineItem data type structure to value.
For details, see "LineItem" in the "Data Definition" chapter.

this.$lineCount Type: Number

The number of lines in the loaded flowchart, initially 0.

this.$areaData Type: JsonMap

Loading of all the grouping lanes in the flowchart Map data set; to lane's idas a key, AreaItem data type structure to value.
For details, see "AreaItem" in the "Data Definition" chapter.

this.$areaCount Type: Number

The number of grouped lanes in the loaded flowchart, initially 0.

this.$max Type: Number

Calculate the SEQUENCE of the suffix sequence number in the default element name. The default value is 1;
the initial value comes from the initNum attribute of the parameters passed in the constructor, or the initNum attribute of the flowchart JSON data is loaded;
when there are elements in the flowchart, When manually added, the value increases by one.

this.$deletedItem Type: JsonMap( {id: String )

It is deleted in the editing operation in the flowchart id-map element set;
to element's idas key，element category name as value(There are three values: node', 'line', 'area').

this.$extra Type: Extra

The additional data map set in the loaded flowchart does not directly participate in the construction of the flowchart, and the output of the flowchart data will be returned as it is. For details, see "Extra" in the "Data Definition" chapter.
