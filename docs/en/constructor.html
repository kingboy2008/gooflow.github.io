Constructors

Methods

There are three constructor calls, their params are the same, and their return value is also the same.

    var demo = GooFlow.init(selector, options);    //Factory mode
    var demo = $.createGooFlow(selector, options); //according to jquery extension method
    var demo = new GooFlow(selector, options);     // native definition of class object method

Personally, it is recommended to use the first (factory mode), compatible with the modular front-end development mode:

GooFlow.init('selector', [options])

Parameters

selector String

The Dom container selector to be rendered as a designer, in the same format as the jQuery selector, requires an id attribute.

Return
