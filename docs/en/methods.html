this.setLineType() Return: void

Reset the type of line.

Parameters: ( id, newType, M )

NameType / FormatDescription
idStringTo operate the line id.
newTypeStringThe new type of connection. There are three types of values: 'sl'straight lines, 'lr'fold lines that move left and right in the 'tb'that middle line can move up and down in the middle.
MNumber An abstract value in pixels.
When type='lr' it is required, the middle line indicates the X-coordinate values with respect to a work area.
When type='tb' it is required, the middle line indicates the Y-coordinate values with respect to the work area.
When type='sl' it is no significance.

this.setLineM() Return: void

Sets the X coordinate of the midline of the polyline (when moving left or right) or Y coordinate (when moving up and down); the straight line does not support this method.

Parameters: ( id, M )

NameType / FormatDescription
idStringTo operate the line id.
MNumberThe new M value. When type='lr'the time line in the middle represents the X coordinate value with respect to the work area; when type='tb' expressed with respect to the middle line Y-coordinate value of the operating time zone.

Transaction Control

The following are options.useOperStack=true method interfaces that exist only when the parameters are initialized.

this.pushOper() Return: void

Methods for internal method invocation only: Add data unit edit operations (add/delete/modify/rename/move/callouts, etc.) in the work area to the entire management stack for undo/redo control; manage stacks Up to 50 history operations records can be placed.

Parameters: ( funcName, paras )

NameType / FormatDescription
funcNameStringThe name of the operation method that was pushed into the management stack.
parasArray[]The parameters passed into the operation method of the management stack are put into an array from front to back.

this.pushExternalOper() Return: void

The method was added to the external transaction GooFlow operand stack instance, after the undo / redo operation can be controlled, it is generally used for the additional information other than the transaction flowchart of editing undo / redo control;
Note: In order for an external method can By undo/redo, you need to add pushExternalOper to the execution method that implements the external method.

Parameters: ( func, jsonPara )

NameType / FormatDescription
funcFunctionExecution method objects that are pushed into the management stack.
jsonParaJson ObjectThis is the only literally-oriented JSON argument or data for an external method. It is a JSON object or array that brings in all the information to be passed.
