事件钩子

顶部栏按钮事件

以下是当工具栏按钮被点击时触发的事件定义(虚函数)，只有初始化GooFlow实例时设置了顶部栏，这些事件钩子才存在。因为可直接用this访问GooFlow实例本身，无传参；用户可自行重定义。

另外，用户还可通过GooFlow实例内置的bindHeadBtnEvent接口，自行扩展另外自定义的头部按钮的点击事件。

这些事件钩子的使用方法： 
    instance.myEvent = function(){
        this;  //GooFlow实例本身
        //TODO: 事件响应的实现
    };

this.onBtnNewClick() 返回值：void

新建流程图按钮事件定义。

this.onBtnOpenClick() 返回值：void

打开流程图按钮事件定义。

this.onBtnSaveClick() 返回值：void

保存流程图按钮事件定义。

this.onFreshClick() 返回值：void

刷新重载流程图按钮事件定义。

this.onPrintClick() 返回值：void

打印机按钮事件定义。

流程图元素事件

以下绑定的是节点/连线/分组泳道的一些操作事件，用户可自行重定义。这些事件可直接通过this访问GooFlow实例本身；并且这些事件触发的方法若返回false将阻止默认操作或事件发生。

这些事件钩子的使用方法： 
    instance.myEvent = function( args[] ){
        this;  //GooFlow实例本身
        //TODO: 事件响应的实现
        return bool; //根据实例情况返回false阻止默认事件，或者返回其它值让默认事件发生
    };

this.onItemFocus() 返回值：boolean

当操作某个元素（节点/连线）被由不选中变成选中时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线两种。

this.onItemBlur() 返回值：boolean

当操作某个元素（节点/连线）被由选中变成不选中时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线两种。

this.onItemMark() 返回值：boolean

当用醒目颜色标注某个元素（节点/连线）时触发的方法。

传参：( id, type, mark )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线两种。
markBoolean是否加标注。是则true，否则false。

this.onItemDbClick() 返回值：boolean

当操作某个元素（节点/连线/分组泳道）被双击时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。

this.onItemRightClick() 返回值：boolean

当操作某个元素（节点/连线/分组泳道）被右键单击时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。

this.onItemAdd() 返回值：boolean

当某个元素（节点/连线/分组泳道）被添加时，触发的方法。

传参：( id, type, json )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。
json[ NodeItem | LineItem | AreaItem ]描述元素的相应结构数据。 当元素是节点时为NodeItem型数据；当元素是连线时为LineItem型数据；当元素是分组泳道时为AreaItem型数据。

this.onItemDel() 返回值：boolean

当某个元素（节点/连线/分组泳道）被删除时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。

this.onItemMove() 返回值：boolean

当某个元素（节点/分组泳道）被移动至新位置时，触发的方法。

传参：( id, type, left, top )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'area'分组泳道两种。
leftNumber元素在新位置上的相对于工作区的左边距，单位像素。
topNumber元素在新位置上的相对于工作区的上边距，单位像素。

this.onItemRename() 返回值：boolean

当某个元素（节点/连线/分组泳道）被重命名时，触发的方法。

传参：( id, name, type )

名称类型/格式描述
idString元素的id，唯一标识。
nameString元素的新名称（name属性的新值）。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。

this.onItemResize() 返回值：boolean

当某个元素（节点/分组泳道）被移动至新位置时，触发的方法。

传参：( id, type, width, height )

名称类型/格式描述
idString元素的id，唯一标识。
typeString元素的种类，有'node'节点、'line'连线、'area'分组泳道三种。
widthNumber元素的新宽度，单位像素；元素为节点时最小104px，元素为分组泳道时最小200px。
heightNumber元素的新高度，单位像素；元素为节点时最小26px，元素为分组泳道时最小100px。

this.onLineMove() 返回值：boolean

当移动某条折线中段的位置时，触发的方法。

传参：( id, M )

名称类型/格式描述
idString连线的id，唯一标识。
MNumber折线中段的新X坐标（当type='lr'时）或者新y坐标（当type='tb'时），单位像素。

this.onLineSetType() 返回值：boolean

当变换某条连线的类型时，触发的方法。

传参：( id, type )

名称类型/格式描述
idString连线的id，唯一标识。
typeString连线的新类型，有三种：'sl'直线；'lr'中段可左右移动的折线'；'tb'中段可上下移动的折线有直线。

this.onLinePointMove() 返回值：boolean

当变换某条连线的端点连接的节点时（即某端点由连着节点A改成连着节点B），触发的方法。

传参：( id, newStart, newEnd )

名称类型/格式描述
idString连线的id，唯一标识。
newStartString新的起始节点的id；如果为null则表示原端点不变。
newEndString新的目标节点的id；如果为null则表示原端点不变。
